# RGPV SGPA & Grade to Marks Calculator

This web application helps RGPV students calculate their SGPA and actual marks
based on grades, credits, and subject type (theory/practical).

## Features
- Convert grades into actual marks
- SGPA calculation for multiple subjects
- Grace marks (D##) support
- Clean and responsive UI
- No popup alerts, inline validation messages

## Tech Stack
- HTML
- CSS
- JavaScript
